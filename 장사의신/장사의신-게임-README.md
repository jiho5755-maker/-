# 🏪 장사의 신 - 창업 컨설팅 시스템

초중고 학생들을 위한 경제 관념 교육용 게임 관리자 프로그램입니다.

## 📋 프로그램 소개

이 프로그램은 학생들이 가상의 시장에서 창업을 체험하면서 경제 개념을 배울 수 있도록 도와주는 관리자용 도구입니다.

### 주요 기능

- **🎯 시장 설정**: 게임의 전체 화폐량과 구매자 수, 구매자 성향 비율 설정
- **👨‍🎓 학생 정보 입력**: 학생 이름과 창업 정보 입력
- **💰 원가 계산**: 시장 상황에 맞는 적정 원가 자동 계산
- **📊 창업 조언**: 선택한 사업 유형과 등급에 따른 맞춤형 조언 제공
- **🔗 Google Sheets 연동**: 여러 명이 PC/모바일에서 실시간 데이터 공유 (NEW!)

## 🚀 설치 방법

### 1. Python 설치 확인

Python 3.8 이상이 설치되어 있어야 합니다.

```bash
python --version
```

### 2. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

이 명령어로 다음 패키지들이 자동 설치됩니다:
- streamlit (웹 앱 프레임워크)
- pandas (데이터 처리)
- openai (AI 조언자)
- gspread (Google Sheets 연동)
- google-auth (Google 인증)

### 3. Google Sheets 연동 설정 (선택사항)

여러 명이 동시에 데이터를 공유하려면 Google Sheets 연동을 설정하세요:

📖 **상세 가이드**: `Google-Sheets-연동-가이드.md` 파일 참고
⚡ **빠른 시작**: `빠른시작-Google-Sheets.md` 파일 참고

Google Sheets를 설정하지 않아도 로컬 모드로 사용할 수 있습니다.

## ▶️ 실행 방법

터미널에서 다음 명령어를 실행하세요:

```bash
streamlit run 장사의신-게임.py
```

웹 브라우저가 자동으로 열리면서 프로그램이 실행됩니다.

## 📖 사용 가이드

### 1단계: 시장 설정 (사이드바)

왼쪽 사이드바에서 게임의 기본 설정을 입력합니다:

- **💰 시장 총 화폐량**: 게임에서 사용할 전체 돈의 양 (예: 1,000,000원)
- **👥 전체 구매자 수**: 조교나 선생님 등 구매자 역할 인원 (예: 30명)
- **🎯 구매자 성향 비율**: 큰손/일반/짠물 비율 (합계 100%)

### 2단계: 학생 정보 입력 (메인 화면)

- **📝 학생 이름**: 창업할 학생의 이름 입력

### 3단계: 창업 유형 선택

5가지 사업 유형 중 선택:

1. **🔨 뚝딱뚝딱** (제조/만들기): 원가 60%, 노력 ⭐⭐⭐⭐
2. **🏃 대신하기** (서비스/몸쓰기): 원가 20%, 노력 ⭐⭐⭐⭐⭐
3. **🛒 골라오기** (유통/떼오기): 원가 50%, 노력 ⭐⭐⭐
4. **📚 알려주기** (지식/정보): 원가 10%, 노력 ⭐⭐
5. **🎪 빌려주기** (대여/공유): 원가 40%, 노력 ⭐

### 4단계: 상품 등급 선택

3가지 등급 중 선택:

- **💚 일반형** (가성비): x0.8 배수, 짠물 고객 타겟
- **💙 고급형** (퀄리티): x1.2 배수, 일반 고객 타겟
- **💜 하이엔드** (명품): x1.6 배수, 큰손 고객 타겟

### 5단계: 결과 확인

"🎉 창업 컨설팅 결과 보기" 버튼을 클릭하면:

- **💰 추천 원가**: 계산된 적정 원가
- **🔥 노력/피로도**: 사업의 힘든 정도 (별점)
- **🎯 타겟 고객**: 공략해야 할 고객층
- **📊 상세 분석**: 시장 정보와 사업 분석
- **🌟 창업 조언**: 맞춤형 조언
- **💵 권장 판매가**: 최소/권장/최대 판매가 가이드

## 🧮 원가 계산 로직

```
1인당 평균 예산 = 시장 총 화폐량 ÷ 구매자 수
1인당 구매 목표 = 4개 (고정)
기준 단가 = 1인당 평균 예산 ÷ 4

추천 원가 = 기준 단가 × 유형별 비율 × 등급별 승수 × 랜덤(0.9~1.1)
```

### 유형별 원가 비율

- 뚝딱뚝딱: 60%
- 골라오기: 50%
- 빌려주기: 40%
- 대신하기: 20%
- 알려주기: 10%

### 등급별 승수

- 일반형: ×0.8
- 고급형: ×1.2
- 하이엔드: ×1.6

## 💡 수업 활용 팁

1. **사전 준비**: 수업 시작 전 시장 설정을 미리 해두세요
2. **개별 컨설팅**: 각 학생이 자신의 아이디어로 창업 유형과 등급을 선택하게 하세요
3. **비교 학습**: 여러 조합을 시도해보며 원가와 마진의 관계를 배울 수 있습니다
4. **실전 적용**: 계산된 원가를 바탕으로 실제 창업 활동을 진행하세요
5. **결과 분석**: 게임 종료 후 누가 가장 많은 이익을 냈는지 분석해보세요
