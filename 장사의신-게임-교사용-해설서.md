## 프로그램 개요

### 교육 목표
- **매출 ≠ 수익** 개념 이해
- 원가의 중요성 체감
- 마진율 계산 능력 향상
- 시장 상황에 따른 전략적 의사결정
- 실전 창업 시뮬레이션

### 게임 구조
1. **창업 컨설팅**: 학생이 사업 유형과 등급 선택 → 원가 산출
2. **1라운드 판매**: 초기 전략으로 판매 진행
3. **중간 분석**: 시장 상황 공유 (리더보드 활용)
4. **2라운드 판매**: 전략 수정 후 재도전
5. **최종 발표**: 매출 → 수익 순위 반전 효과

## 원가 산정 로직 상세 해설

### 1단계: 기준 단가 계산

```
1인당 평균 예산 = 시장 총 화폐량 ÷ 구매자 수
1인당 구매 목표 = 4개 (고정)
기준 단가 = 1인당 평균 예산 ÷ 4
```

**💡 왜 4개로 고정했나요?**

1. **게임 밸런스**: 모든 학생이 동일한 기준으로 경쟁
2. **현실성**: 초등학생이 실제로 살 수 있는 가격대 형성 (개당 5,000~10,000원)
3. **교육 효과**: "한정된 예산으로 여러 가지 구매" 경험
4. **운영 편의**: 시장 총액만 조절하면 자동으로 밸런스 유지

> 💭 **디자인 철학**: 변수를 최소화하여 교사가 쉽게 운영하고, 학생은 순수하게 "전략"에 집중할 수 있도록 설계했습니다.

**예시:**
- 시장 총액: 1,000,000원
- 구매자 수: 30명
- 1인당 평균 예산 = 1,000,000 ÷ 30 = 33,333원
- 기준 단가 = 33,333 ÷ 4 = **8,333원** ← 현실적인 학용품/간식 가격!

### 2단계: 유형별 원가율 적용

```
유형별 원가 = 기준 단가 × 유형별 원가율
```

**유형별 원가율:**
- 🔨 뚝딱뚝딱 (제조): **60%** - 재료비가 많이 듦
- 🛒 골라오기 (유통): **50%** - 도매가 + 배송비
- 🎪 빌려주기 (대여): **40%** - 초기 구매 비용
- 🏃 대신하기 (서비스): **20%** - 인건비 위주
- 📚 알려주기 (지식): **10%** - 거의 원가 없음
- ⚙️ 직접 설정: **5~80%** - 학생이 조정 가능

**예시 (제조업):**
- 유형별 원가 = 8,333 × 0.6 = **5,000원**

### 3단계: 등급별 승수 적용

```
등급 적용 원가 = 유형별 원가 × 등급별 승수
```

**등급별 승수:**
- 💚 일반형 (가성비): **×0.8** - 저렴한 재료 사용
- 💙 고급형 (퀄리티): **×1.2** - 좋은 재료 사용
- 💜 하이엔드 (명품): **×1.6** - 최고급 재료

**예시 (고급형):**
- 등급 적용 원가 = 5,000 × 1.2 = **6,000원**

### 4단계: 랜덤성 부여

```
최종 원가 = 등급 적용 원가 × 랜덤 계수 (0.9 ~ 1.1)
```

**이유:**
- 같은 선택을 해도 약간의 차이 발생 (실제 시장처럼)
- 학생들의 흥미 유발

**예시:**
- 랜덤 계수 = 1.05
- 최종 원가 = 6,000 × 1.05 = **6,300원**

## 창업 유형별 특성

### 🔨 뚝딱뚝딱 (제조/만들기)
- **원가율**: 60% (높음)
- **노력도**: ⭐⭐⭐⭐ (4/5)
- **특징**: 재료비가 많이 들지만, 직접 만들어 독창성 가능
- **예시**: 팔찌, 쿠키, 비누, 열쇠고리, 인형
- **교육 포인트**: 제조업의 원가 구조, 품질 관리의 중요성

### 🏃 대신하기 (서비스/몸쓰기)
- **원가율**: 20% (낮음)
- **노력도**: ⭐⭐⭐⭐⭐ (5/5)
- **특징**: 원가는 낮지만 체력 소모가 큼
- **예시**: 심부름, 신발끈 묶어주기, 청소, 짐 나르기
- **교육 포인트**: 노동의 가치, 시간과 체력도 원가

### 🛒 골라오기 (유통/떼오기)
- **원가율**: 50% (중간)
- **노력도**: ⭐⭐⭐ (3/5)
- **특징**: 중간 원가, 물건 선택이 중요
- **예시**: 문구류, 간식, 장난감 도매
- **교육 포인트**: 유통 마진, 중간 상인의 역할

### 📚 알려주기 (지식/정보)
- **원가율**: 10% (매우 낮음)
- **노력도**: ⭐⭐ (2/5)
- **특징**: 거의 원가가 없지만, 가치 있는 정보가 필요
- **예시**: 게임 공략, 과외, 꿀팁 공유
- **교육 포인트**: 지식의 가치, 무형 상품

### 🎪 빌려주기 (대여/공유)
- **원가율**: 40% (중하)
- **노력도**: ⭐ (1/5)
- **특징**: 초기 투자 필요하지만 반복 사용 가능
- **예시**: 보드게임, 운동용품, 악기, 책
- **교육 포인트**: 공유경제, 자산 활용

### ⚙️ 직접 설정하기
- **원가율**: 5~80% (학생 조정)
- **노력도**: 1~5 (학생 조정)
- **특징**: 창의적인 아이디어를 수치화
- **예시**: 독특한 융합 사업
- **교육 포인트**: 사업 계획의 중요성

## 상품 등급별 전략

### 💚 일반형 (가성비) - ×0.8
- **원가**: 낮음
- **타겟**: 🤏 짠물 고객 (예산의 50%)
- **전략**: 대량 판매, 빠른 회전율
- **예시**: 1000원짜리 팔찌 100개
- **장점**: 많은 사람에게 팔 수 있음
- **단점**: 마진율이 낮아 많이 팔아야 함

### 💙 고급형 (퀄리티) - ×1.2
- **원가**: 중간
- **타겟**: 😊 일반 고객 (예산의 100%)
- **전략**: 균형잡힌 판매, 안정적
- **예시**: 3000원짜리 쿠키 30개
- **장점**: 가장 무난하고 안전한 선택
- **단점**: 차별화가 어려움

### 💜 하이엔드 (명품) - ×1.6
- **원가**: 높음
- **타겟**: 🤑 큰손 고객 (예산의 200%)
- **전략**: 소량 판매, 높은 마진
- **예시**: 10000원짜리 고급 인형 5개
- **장점**: 적게 팔아도 큰 수익
- **단점**: 판매가 안되면 재고 부담

## 판매가 책정 가이드

### 권장 판매가 계산

프로그램이 제시하는 권장가:

```
최소가 = 원가 × 1.2  (20% 마진)
권장가 = 원가 × 1.5  (50% 마진)
최대가 = 원가 × 2.0  (100% 마진)
```

**예시 (원가 5,000원):**
- 🥉 최소가: 6,000원 (이익 1,000원)
- 🥈 권장가: 7,500원 (이익 2,500원)
- 🥇 최대가: 10,000원 (이익 5,000원)

### 판매가 전략 교육

**너무 싸게 팔면:**
- ✅ 많이 팔 수 있음
- ❌ 이익이 적어 총 수익 낮음
- ❌ 원가도 못 건지면 적자

**적정가로 팔면:**
- ✅ 적당한 판매량
- ✅ 적당한 이익
- ✅ 가장 안정적

**너무 비싸게 팔면:**
- ✅ 하나 팔 때마다 큰 이익
- ❌ 판매량 급감
- ❌ 재고 부담

## 핵심 교육 포인트

### 1. 매출 ≠ 수익 (가장 중요!)

**매출 (Revenue):**
```
매출 = 판매가 × 판매량
```
- 얼마나 많이 팔았는가
- 겉으로 보이는 돈

**수익 (Profit):**
```
수익 = 매출 - 원가
    = (판매가 - 원가) × 판매량
```
- 실제로 남은 돈
- 진짜 벌어들인 돈

**반전 효과 활용:**
1. 먼저 매출 순위 발표 → "OOO이가 1등!"
2. 수익 순위 발표 → 순위 변동!
3. **깨달음**: 많이 팔았어도 원가가 높으면 수익 적음

**실제 사례:**
- A학생: 1000원에 100개 판매 = 매출 100,000원 (원가 900원 → 수익 10,000원)
- B학생: 5000원에 10개 판매 = 매출 50,000원 (원가 2000원 → 수익 30,000원)
- **결과**: 매출은 A가 2배인데 수익은 B가 3배!

### 2. 마진율의 의미

```
마진율 = (수익 ÷ 매출) × 100
```

- 10% 이하: 거의 남는 게 없음
- 20~30%: 최소한의 이익
- 40~60%: 적정 마진
- 70% 이상: 매우 높은 마진

### 3. 가격 탄력성

- **저가 전략**: 많이 팔아야 수익 (노동집약적)
- **고가 전략**: 적게 팔아도 수익 (브랜드 가치)
- **중가 전략**: 균형잡힌 접근 (안정적)

### 4. 재고 리스크

- 팔리지 않은 상품 = 묶인 돈
- 원가 회수 못하면 = 손해
- 적정 생산량 중요

## 수업 진행 시나리오

### 사전 준비 (수업 1주일 전)

1. **시장 설정 결정**
   - 학급 인원에 맞춰 시장 총액 설정
   - 조교/선생님 구매자 수 확인
   - 구매자 성향 비율 결정

2. **예시 계산**
   - 몇 가지 유형으로 직접 계산해보기
   - 학생들에게 설명할 예시 준비

3. **준비물**
   - 컴퓨터 + 프로젝터 (리더보드용)
   - 학생들이 만들 재료들
   - 가짜 화폐 (선택사항)

### 1차시: 창업 컨설팅 (40분)

**도입 (10분)**
- 프로그램 소개
- 게임 규칙 설명
- 매출 vs 수익 개념 설명

**전개 (25분)**
- 각 학생 창업 컨설팅 받기
- 사업 유형과 등급 선택
- 원가 확인 및 판매가 결정
- 상품 준비 시작

**정리 (5분)**
- 다음 시간 안내
- 준비 독려

### 2차시: 1라운드 판매 (40분)

**도입 (5분)**
- 시장 상황 안내
- 판매 시작 선언

**전개 (30분)**
- 실제 판매 활동
- 구매자(조교)들의 구매
- 판매 결과 실시간 입력

**정리 (5분)**
- 1라운드 종료
- 데이터 저장 확인

### 3차시: 중간 분석 및 전략 수정 (40분)

**도입 (10분)**
- 1라운드 결과 공유
- 매출 순위 발표 (리더보드 활용)

**전개 (25분)**
- 시장 상황 분석
- 전략 토론 (모둠별)
- 판매가 조정 결정
- 2라운드 준비

**정리 (5분)**
- 조정 전략 발표
- 다음 시간 안내

### 4차시: 2라운드 판매 (40분)

**도입 (5분)**
- 수정 전략 확인
- 2라운드 시작

**전개 (30분)**
- 실제 판매 활동
- 판매 결과 입력

**정리 (5분)**
- 2라운드 종료
- 최종 발표 예고

### 5차시: 최종 발표 및 반성 (40분)

**도입 (5분)**
- 전체 결과 집계 확인

**전개 (25분)**
1. **1단계**: 💰 매출 순위 발표 (10분)
   - "OOO이가 매출 1등!"
   - 축하와 칭찬
   
2. **2단계**: 💎 수익 순위 발표 (15분)
   - 순위 변동 확인
   - 반전의 순간!
   - 진짜 1등 발표
   - **핵심 질문**: "왜 매출 1등이 수익 1등이 아닐까?"

**정리 (10분)**
- 배운 점 발표
- 원가의 중요성 강조
- 실생활 적용 방안 토론

## 자주 묻는 질문 (FAQ)

### Q1. 원가가 너무 높게 나왔어요. 조정할 수 있나요?

**A:** 원가는 자동 계산되지만, 학생이 다시 컨설팅을 받을 때 다른 유형이나 등급을 선택하면 달라집니다. 또는 "⚙️ 직접 설정하기"를 사용해 원가율을 조정할 수 있습니다.

### Q2. 학생이 판매가를 너무 싸게/비싸게 책정했어요.

**A:** 그대로 진행하세요! 이것이 바로 교육의 기회입니다. 결과를 보고 학생 스스로 배우는 것이 중요합니다.

### Q3. 재고가 남았는데 원가 처리는 어떻게 하나요?

**A:** 프로그램은 "판매한 수량"만 계산합니다. 재고는 손실로 처리되며, 이것도 중요한 교육 포인트입니다.

### Q4. 구매자 성향 비율을 어떻게 설정하는 게 좋나요?

**A:** 
- 균형잡힌 시장: 큰손 20% / 일반 50% / 짠물 30%
- 부유한 시장: 큰손 40% / 일반 40% / 짠물 20%
- 저소득 시장: 큰손 10% / 일반 30% / 짠물 60%

### Q5. 시장 총액을 얼마로 설정하면 좋나요?

**A:** 
- 학급 인원 × 33,000원 정도 추천
- 예: 30명 → 1,000,000원
- 너무 작으면: 가격이 낮아져 계산 어려움
- 너무 크면: 가격이 높아져 비현실적

### Q6. 한 학생이 여러 유형을 동시에 할 수 있나요?

**A:** 프로그램상으로는 한 유형만 등록되지만, 교육적으로는 가능합니다. 단, 데이터 입력 시 주 사업 하나만 기록하세요.

### Q7. 판매 데이터 입력을 학생이 직접 해도 되나요?

**A:** 가능하지만, 조작 가능성이 있으므로 교사나 보조 교사가 입력하는 것을 권장합니다.

### Q8. 프로그램 없이 손으로 계산해도 되나요?

## 부록: 계산 예시 전체 과정

### 상황 설정
- 시장 총액: 1,000,000원
- 구매자 수: 30명
- 구매자 성향: 큰손 20%(6명) / 일반 50%(15명) / 짠물 30%(9명)

### 학생 A: 제조업 + 고급형

**1단계: 기준 단가**
```
1인당 평균 예산 = 1,000,000 ÷ 30 = 33,333원
기준 단가 = 33,333 ÷ 4 = 8,333원
```

**2단계: 유형별 원가 (제조 60%)**
```
유형별 원가 = 8,333 × 0.6 = 5,000원
```

**3단계: 등급별 승수 (고급형 1.2)**
```
등급 원가 = 5,000 × 1.2 = 6,000원
```

**4단계: 랜덤 (예: 1.05)**
```
최종 원가 = 6,000 × 1.05 = 6,300원
```

**판매 시나리오:**
- 판매가: 10,000원
- 판매량: 20개
- 매출 = 10,000 × 20 = **200,000원**
- 총원가 = 6,300 × 20 = 126,000원
- 수익 = 200,000 - 126,000 = **74,000원**
- 마진율 = 74,000 ÷ 200,000 = **37%**

### 학생 B: 서비스업 + 일반형

**1~4단계 계산:**
```
기준: 8,333원
유형 (20%): 8,333 × 0.2 = 1,667원
등급 (0.8): 1,667 × 0.8 = 1,333원
랜덤 (1.1): 1,333 × 1.1 = 1,466원
```

**판매 시나리오:**
- 판매가: 3,000원
- 판매량: 50개
- 매출 = 3,000 × 50 = **150,000원**
- 총원가 = 1,466 × 50 = 73,300원
- 수익 = 150,000 - 73,300 = **76,700원**
- 마진율 = 76,700 ÷ 150,000 = **51%**

**결과 분석:**
- 매출: A(200,000) > B(150,000)
- **수익: B(76,700) > A(74,000)** ← 반전!
- B가 매출은 적지만 원가율이 낮아 수익이 더 많음!
