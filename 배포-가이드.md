# 🌐 장사의 신 게임 - 온라인 배포 가이드

## 방법 1: ngrok 사용하기 (가장 빠름) ⚡

### 1단계: ngrok 계정 만들기
1. https://dashboard.ngrok.com/signup 접속
2. 구글 계정으로 간편 가입
3. 무료로 사용 가능!

### 2단계: authtoken 받기
1. https://dashboard.ngrok.com/get-started/your-authtoken 접속
2. 표시된 토큰 복사

### 3단계: 토큰 설정
터미널에서 실행:
```bash
cd "/Users/jangjiho/Desktop/커서AI"
source .venv/bin/activate
ngrok config add-authtoken YOUR_TOKEN_HERE
```

### 4단계: 서버 시작
```bash
# 터미널 1: Streamlit 실행
cd "/Users/jangjiho/Desktop/커서AI"
source .venv/bin/activate
streamlit run 장사의신-게임.py

# 터미널 2: ngrok 실행
cd "/Users/jangjiho/Desktop/커서AI"
source .venv/bin/activate
ngrok http 8502
```

✅ ngrok이 `https://xxxxxx.ngrok.io` 같은 URL을 제공합니다!

## 방법 2: Streamlit Community Cloud (무료 영구 호스팅) 🎁

### 장점
- 24시간 온라인 유지
- 무료 도메인 제공
- 코드 변경 시 자동 업데이트
- ngrok보다 더 안정적

### 1단계: GitHub에 코드 업로드
```bash
cd "/Users/jangjiho/Desktop/커서AI"

# Git 초기화 (처음만)
git init
git add 장사의신-게임.py requirements.txt
git commit -m "장사의 신 게임 추가"

# GitHub 저장소 만들고 푸시
# (GitHub에서 새 저장소 생성 후)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

### 2단계: Streamlit Cloud 배포
1. https://share.streamlit.io 접속
2. GitHub 계정으로 로그인
3. "New app" 클릭
4. Repository 선택
5. Main file path: `장사의신-게임.py` 입력
6. Deploy!

✅ 몇 분 후 `https://your-app.streamlit.app` 같은 영구 URL을 받습니다!
