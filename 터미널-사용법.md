# 🖥️ 터미널 사용법 - 초보자 완전 가이드

## 📍 터미널이란?

컴퓨터에 **글자로 명령**을 내리는 프로그램입니다.
마우스 대신 키보드로 명령어를 입력해요!

---

## 🚀 빠른 시작 (3단계)

### 1단계: 터미널 열기

**⌘ (Command) + Space** → "터미널" 입력 → Enter

또는

**Cursor 하단** → "터미널" 탭 클릭

---

### 2단계: 명령어 복사

아래 명령어를 **마우스로 드래그**해서 선택 후 **⌘+C** (복사):

```bash
cd "/Users/jangjiho/Desktop/커서AI" && ./배포하기.sh
```

---

### 3단계: 터미널에 붙여넣기

터미널 화면 아무 곳이나 클릭 후 **⌘+V** (붙여넣기) → **Enter** 키

완료! 🎉

---

## 🎯 Cursor에서 터미널 사용하기 (더 쉬움!)

Cursor 에디터 안에 터미널이 있어요!

### 방법 1: Cursor 내장 터미널 사용 (추천) ⭐

1. **Cursor 화면 하단 보기**
2. **"터미널" 탭 클릭** (없으면 다음 방법)
3. 또는 **상단 메뉴**: View → Terminal
4. 또는 단축키: **⌃ (Control) + `** (백틱, ESC 아래 키)

👉 하단에 터미널이 열립니다!

### 여기에 명령어 입력:

```bash
cd "/Users/jangjiho/Desktop/커서AI"
./배포하기.sh
```

## ⌨️ 명령어 입력 단계별 가이드

### 실제로 해보기:

**1단계**: Cursor 하단 터미널에서 클릭 (깜빡이는 커서 확인)

**2단계**: 아래 명령어 **복사** (⌘+C):
```bash
cd "/Users/jangjiho/Desktop/커서AI"
```

**3단계**: 터미널에 **붙여넣기** (⌘+V)

**4단계**: **Enter** 키 누르기

**5단계**: 다음 명령어 입력:
```bash
./배포하기.sh
```

**6단계**: **Enter** 키 누르기

완료! 🎊

## 💡 터미널 기본 용어

| 용어 | 의미 | 예시 |
|------|------|------|
| **프롬프트** | 명령어 입력 대기 상태 | `~ %` |

## 🔧 자주 사용하는 명령어

### 기본 명령어:

```bash
# 현재 위치 확인
pwd

# 폴더 이동
cd /경로/폴더명

# 폴더 내용 보기
ls

# 화면 지우기

```

```bash

source .venv/bin/activate
streamlit run 장사의신-게임.py
```

## 🆘 문제 해결

### "command not found" 에러
→ 명령어 오타 확인 또는 올바른 폴더에 있는지 확인

```bash
# 현재 위치 확인
pwd

# 올바른 폴더로 이동
cd "/Users/jangjiho/Desktop/커서AI"
```

### "Permission denied" 에러
→ 실행 권한 부여

```bash
chmod +x 배포하기.sh
```

### 터미널이 멈춘 것 같아요
→ **⌃ (Control) + C** 누르면 중단됨
