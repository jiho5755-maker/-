===================================================
⭐⭐ 최종 통합본 - 초간단 설치 (3분) ⭐⭐
===================================================

하단 레이아웃 + 채널톡 스크립트 모두 포함!
플러그인 키까지 입력 완료!

===================================================
준비 사항
===================================================

1. 채널톡 관리자에서 "버튼 숨기기" ON ✅
   (설정 > 버튼 설치 및 설정 > 버튼 숨기기)

2. 이 파일:
   ⭐통합본-채널톡포함.html

===================================================
STEP 1: 파일 복사 (1분)
===================================================

1. ⭐통합본-채널톡포함.html 파일 열기

2. Cmd+A (전체 선택)

3. Cmd+C (복사)

===================================================
STEP 2: 메이크샵 설치 (1분)
===================================================

1. 메이크샵 관리자 로그인

2. 쇼핑몰 디자인 관리 > 디자인 설정

3. [하단 공통 레이아웃] 클릭

4. 기존 내용 전체 삭제

5. Cmd+V (붙여넣기)

6. [저장] 클릭

===================================================
STEP 3: 확인 (1분)
===================================================

1. 쇼핑몰 메인 페이지 접속

2. Cmd+Shift+R (강력 새로고침)

3. 확인:
   ✅ 세이지 그린 ? 버튼 보임
   ✅ 채널톡 기본 버튼 안 보임

4. ? 버튼 클릭

5. [상담원 연결하기] 클릭

6. 🎉 채널톡 메신저 열림! 🎉

===================================================
추가 확인 (개발자 도구)
===================================================

만약 작동 안 하면:

1. F12 (개발자 도구)

2. Console 탭

3. 입력:
   window.ChannelIO

4. 결과:
   ✅ function 나옴 → 정상!
   ❌ undefined 나옴 → 저장 다시 확인

===================================================
채널톡 관리자 설정 (필수!)
===================================================

⚠️ 중요: 반드시 채널톡 관리자에서 설정!

채널톡 관리자 로그인
  ↓
좌측 [설정] 클릭
  ↓
[버튼 설치 및 설정]
  ↓
"버튼 숨기기" 토글 ON ✅
  ↓
자동 저장됨

이걸 안 하면 버튼이 2개 보입니다!

===================================================
문제 해결
===================================================

Q. 메신저가 안 열려요
A1. 채널톡 관리자 > 버튼 숨기기 ON 확인
A2. F12 > Console > window.ChannelIO 확인
A3. 페이지 Cmd+Shift+R 새로고침

Q. 저장이 안 돼요
A. 브라우저 변경 (Chrome → Edge)
   메이크샵 고객센터: 1600-5455

Q. 채널톡 버튼이 2개 보여요
A. 채널톡 관리자 > 버튼 숨기기 ON
