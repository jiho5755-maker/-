1. ⭐⭐⭐프로-워크플로우.html 파일 열기
2. 전체 선택 (Ctrl+A 또는 Cmd+A)
3. 복사 (Ctrl+C 또는 Cmd+C)
4. 메이크샵 관리자 페이지 로그인
5. [쇼핑몰 디자인] → [디자인 설정(D4)]

7. 기존 코드 전체 삭제
8. 붙여넣기 (Ctrl+V 또는 Cmd+V)
9. [저장] 버튼 클릭
10. 쇼핑몰 메인 페이지에서 ? 버튼 확인

   (또는 브라우저에서 직접 열기)

2. 즐겨찾기에 추가 (Ctrl+D 또는 Cmd+D)

3. 실시간으로 고객 문의 확인 가능!

(localStorage는 브라우저별로 독립적)

   → 다른 페이지로 이동

   → [새 대화] 버튼 클릭

   • 평균 만족도
   • 활성 세션 수

   • 최신순 정렬
   • 유형별 색상 구분
   • 클릭하면 상세 내용 펼침

4단계: 상세 정보 확인
   • 주문번호
   • 연락처
   • 사유
   • 대화 내역 (최대 10개 메시지)

5단계: 필터링 & 검색
   • 유형별 필터
   • 날짜별 필터
   • 세션 ID 검색

6단계: 다운로드
   • 개별 문의: [다운로드] 버튼
   • 전체 문의: [전체 내역 다운로드]
   • TXT 파일로 저장

7단계: 데이터 관리
   • 개별 삭제: [삭제] 버튼
   • 전체 삭제: [전체 삭제] 버튼

통계 제공:
• 전체 문의 건수
• 오늘 문의 건수
• 평균 만족도 (5점 만점)
• 활성 세션 수

필터링:
• 유형별: 반품/교환, 적립금, 일반, 만족도
• 날짜별: 특정 날짜 선택
• 세션별: 세션 ID로 검색
