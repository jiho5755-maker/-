## 📦 포함된 기능

### 1️⃣ 신규 방문자 환영 팝업
```
첫 방문 감지 → 환영 팝업 표시
"첫 구매 3,000원 적립금 증정!"
이메일 수집 → 자동 쿠폰 발송
```

### 2️⃣ 회원 등급별 맞춤 팝업
```
일반 회원: 기본 할인 쿠폰
VIP 회원: 전용 20% 할인
비회원: 회원가입 유도
```

### 3️⃣ 이탈 방지 팝업
```
마우스 이탈 감지 → 팝업 표시
"잠깐! 5% 할인 쿠폰 드립니다"
장바구니 상품 있을 때 우선 표시
```

### 4️⃣ 시간대별 자동 팝업
```
점심 시간: "런치 타임 특가!"
저녁 시간: "오늘의 베스트 상품"
주말: "주말 특별 할인"
```

## 🔍 문제 해결

### Q. 팝업이 계속 뜨는데요?
A. 쿠키 확인: 개발자 도구 > Application > Cookies

### Q. 특정 페이지에만 표시하고 싶어요
A. URL 조건 추가:
```javascript
if(location.pathname === '/product/') {
  // 상품 페이지에만 표시
}
```

### Q. 모바일에서 너무 커요
A. CSS 미디어 쿼리로 조정:
```css
@media (max-width: 768px) {
  .mk-modal { width: 90%; }
}
```
