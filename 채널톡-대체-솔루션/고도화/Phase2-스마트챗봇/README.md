## 📦 포함된 기능

### 1️⃣ 키워드 자동 인식
```
사용자 입력: "배송"
→ 자동 답변: 배송 안내 + 주문 조회 링크

사용자 입력: "반품"
→ 자동 답변: 반품 절차 + 신청서 링크

사용자 입력: "교환"
→ 자동 답변: 교환 방법 + CS 센터 연결
```

### 2️⃣ 단계별 질문 트리
```
"무엇을 도와드릴까요?"
  ├─ 주문/배송
  │  ├─ 주문 조회
  │  ├─ 배송 추적
  │  └─ 배송지 변경
  ├─ 교환/반품
  │  ├─ 반품 신청
  │  ├─ 교환 신청
  │  └─ 환불 문의
  └─ 기타
     ├─ 회원 정보
     ├─ 적립금
     └─ 쿠폰
```

### 3️⃣ 영업시간 외 자동 응답
```
평일 09:00-18:00: 실시간 상담 가능
그 외 시간: 자동 응답 + 이메일 수집

"현재 영업시간이 아닙니다.
이메일을 남겨주시면 영업일 기준 24시간 내 답변드립니다."
```

### 4️⃣ FAQ 검색 기능
```
검색창: "적립금"
→ 관련 FAQ 자동 표시
→ 찾는 답변 없으면 상담원 연결
```
