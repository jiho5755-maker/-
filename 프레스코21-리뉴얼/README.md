## 📋 프로젝트 개요

프레스코21(www.foreverlove.co.kr)의 전면 리뉴얼 프로젝트입니다.  
**장인 정신**과 **교육형 커머스**를 결합하여, 4060 세대도 쉽게 사용할 수 있는  
프리미엄 공예 쇼핑몰을 구축했습니다.

### 🎯 핵심 가치

1. **장인의 헤리티지**: 이진선 대표의 40년 노하우 전면 배치
2. **교육형 커머스**: 유튜브 가이드 영상과 상품 완벽 연동
3. **B2B 상생**: 강사/협회 회원 전용 도매 시스템
4. **커뮤니티 강화**: 고객 작품 갤러리로 소속감 증대
5. **4060 최적화**: 큰 글씨, 넓은 터치 영역, 명확한 피드백

### 2️⃣ 메인 페이지: 장인 정신의 시각화

**파일:** `02-main-page.html`

#### 🏛️ 이진선 헤리티지 섹션
- 왼쪽: 대표님 작업 영상 (자동 재생)
- 오른쪽: 인터랙티브 타임라인
  - 1982년: 압화 예술의 시작
  - 1999년: 프레스코21 설립
  - 2008년: 전국 강사 양성
  - 2025년: 디지털 교육 혁신

#### 🎨 장르 선택 탭
4가지 공예 장르 (압화/레진/하바리움/부케기프트)
- 탭 클릭 시 부드러운 애니메이션 전환
- 각 장르별 입문 영상 + 추천 세트 상품
- 난이도별 필터링 (초급/중급/고급)
- "영상 속 재료 한 번에 담기" 버튼

#### 💡 큐레이션 시스템
- **취미로 시작하기**: 초보자 입문 세트
- **강사로 성장하기**: 자격증 과정 안내
- **사업으로 확장하기**: 공방 창업 지원

**미리보기:**
```
[헤리티지 영상] + [타임라인]
         ↓
[장르 탭: 압화 | 레진 | 하바리움 | 부케]
         ↓
[유튜브 영상] ←→ [재료 목록 + 난이도 필터]
         ↓
[큐레이션 3종: 취미/강사/사업]
```

### 3️⃣ 상품 상세: 콘텐츠 커머스의 완성

**파일:** `03-product-detail.html`

#### 📽️ 교육형 커머스 레이아웃
```
[유튜브 가이드 영상 (60%)]  |  [영상 속 재료 목록 (40%)]
     - 타임라인 버튼           |    - 타임라인 배지 클릭 → 영상 이동
     - 챕터 네비게이션         |    - 체크박스로 선택/해제
                              |    - 수량 조절
                              |    - 실시간 가격 계산
```

#### 💡 이진선 대표 한 끝 노하우 박스
- 그라데이션 배경에 흰색 글씨
- 프로필 사진 + 40년 경력 뱃지
- 실전 팁 하이라이트 강조

#### 📦 스마트 재료 목록
- 각 재료마다 "타임라인 배지" 표시 (예: 02:45)
- 클릭 시 유튜브 영상의 해당 시간대로 이동
- 체크박스로 필요한 재료만 선택 가능
- 총 금액 자동 계산

**핵심 코드:**
```javascript
// 타임라인 버튼 클릭 시 유튜브 이동
function seekToTime(seconds) {
  player.seekTo(seconds);
  player.playVideo();
}
```

### 4️⃣ B2B 회원 시스템: 강사와 협회 상생

**파일:** `04-b2b-member-system.js`

#### 회원 등급별 혜택

| 등급 | 할인율 | 전용 기능 |
|------|--------|-----------|
| 일반 | 0% | - |
| 강사 | 25% | 도매 발주, 우선 배송 |
| 협회 | 30% | 전용 상담, 이벤트 우선권 |
| VIP | 35% | 최고 혜택 |

#### 자동 UI 변경
- **상단 알림 바**: "⭐ 홍길동님 환영합니다! | 강사 회원 특별가 25% 할인 적용 중"
- **가격 표시**: 일반가 취소선 처리 + 할인가 강조
- **도매 발주 버튼**: 강사/협회 등급만 표시
- **인증 배지**: 우측 하단에 "이진선 장인의 공식 파트너" 뱃지

#### 도매 발주 퀵 리스트
```
[상품명]  [수량 입력]  [단가]  [합계]
압화 키트    10개      26,250원  262,500원
프레싱 툴     5개      11,250원   56,250원
--------------------------------
총 발주 금액               318,750원
```

**테스트 모드:**
```javascript
// 브라우저 콘솔에서
DevMode.setTestMember('instructor'); // 강사 회원으로 테스트
location.reload();
```
