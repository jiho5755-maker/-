## ⏱️ 5분 만에 이해하기

### 우리가 만든 것

**기존 문제점:**
- 장인 정신이 잘 드러나지 않음
- 상품만 나열되어 있어 구매 확신이 낮음
- 강사/협회 회원 전용 기능 부재
- 커뮤니티 요소 부족

**리뉴얼 솔루션:**

1. **메인 페이지** → 이진선 대표 헤리티지 + 장르별 입문 가이드
2. **상품 상세** → 유튜브 영상 + 재료 목록 완벽 연동
3. **B2B 시스템** → 강사/협회 회원 자동 할인가 표시
4. **파트너 지도** → 전국 공방 찾기 (네이버 지도)
5. **작품 갤러리** → 고객 작품 공유 커뮤니티

## 🎯 우선순위별 적용 가이드

### 🔥 최우선 (필수, 1주일 내)

#### 1. 브랜드 디자인 시스템 적용
- **파일:** `01-brand-system.css`
- **작업:** 메이크샵 css 폴더에 업로드
- **효과:** 전체 사이트 디자인 통일

#### 2. 메인 페이지 리뉴얼
- **파일:** `02-main-page.html`
- **준비물:**
  - [ ] 이진선 대표 작업 영상 (1-2분)
  - [ ] 장르별 입문 영상 (압화/레진/하바리움/부케)
- **작업:** 메인화면 관리에서 HTML 적용

#### 3. 회원 등급 설정
- **작업:** 메이크샵 회원등급 관리
- **설정:**
  - 강사: 25% 할인
  - 협회: 30% 할인
  - VIP: 35% 할인

---

### ⭐ 중요 (2주 내)

#### 4. 상품 상세 페이지 유튜브 연동
- **파일:** `03-product-detail.html`
- **준비물:**
  - [ ] 주요 상품별 가이드 영상 촬영
  - [ ] 영상별 타임라인 작성 (00:00, 02:45, 07:00...)
- **작업:** 상품 편집 시 유튜브 영상 ID 입력

#### 5. B2B 시스템 활성화
- **파일:** `04-b2b-member-system.js`
- **작업:** 메이크샵 js 폴더 업로드 후 연동

## 🎬 콘텐츠 제작 가이드

### 영상 촬영 체크리스트

#### 📹 이진선 대표 헤리티지 영상 (필수)
- **길이:** 1-2분
- **내용:**
  - 작업하는 모습 (클로즈업)
  - 재료 다루는 손동작
  - 완성된 작품들
- **분위기:** 고요하고 우아한 BGM
- **용도:** 메인 페이지 자동 재생

#### 📹 장르별 입문 가이드 (필수)

**1) 압화 입문 가이드 (10-15분)**
```
00:00 - 인트로 및 오늘의 재료 소개
02:45 - 꽃 선택과 전처리 (물기 제거 팁!)
07:00 - 프레싱 툴 사용법
12:00 - 건조 시간과 보관 방법
16:00 - 액자에 구도 잡고 완성하기

**2) 레진 입문 가이드 (10-15분)**
```
00:00 - 레진 재료 소개
03:00 - 몰드 선택과 준비
06:00 - 레진액 계량 및 혼합
10:00 - 기포 제거 노하우
14:00 - 경화 및 마감 처리

**3) 하바리움 입문 (10-15분)**
