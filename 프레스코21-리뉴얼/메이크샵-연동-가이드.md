## 2. 파일 업로드 방법

### 2.1 CSS 파일 업로드

**경로:** `스킨폴더 > css > `

업로드할 파일:
```
01-brand-system.css
```

**적용 방법:**

메인 레이아웃 파일(`layout.html` 또는 `main.html`)의 `<head>` 태그 안에 추가:

```html
<link rel="stylesheet" href="/css/01-brand-system.css">
```

### 2.2 JavaScript 파일 업로드

**경로:** `스킨폴더 > js > `

업로드할 파일:
```
04-b2b-member-system.js
```

**적용 방법:**

레이아웃 파일의 `</body>` 태그 직전에 추가:

```html
<script src="/js/04-b2b-member-system.js"></script>
```

### 2.3 HTML 페이지 적용

#### 메인 페이지 (02-main-page.html)
```
관리자 > 디자인 > 메인화면 관리
```
- `02-main-page.html`의 내용을 복사
- 메인화면 HTML 편집기에 붙여넣기
- 기존 메이크샵 변수와 호환되도록 조정

#### 상품 상세 페이지 (03-product-detail.html)
```
관리자 > 디자인 > 스킨관리 > 상품 상세 페이지 편집
```
- `03-product-detail.html`의 구조를 참고
- 메이크샵 상품 변수(`{$product_name}` 등)와 통합

#### 커뮤니티 갤러리 (06-community-gallery.html)
```
관리자 > 게시판 관리 > 게시판 추가
```
- "작품 갤러리" 게시판 생성
- 갤러리형 레이아웃 선택
- 커스텀 HTML로 `06-community-gallery.html` 적용

## 3. 회원 등급 시스템 연동

### 3.1 메이크샵 담당자에게 문의할 내용

메이크샵 고객센터 (1544-3651)에 다음 내용을 문의하세요:

```
안녕하세요, 프레스코21입니다.
회원 등급별 차등 가격 표시를 위해 다음 정보가 필요합니다:

1. 현재 로그인한 회원의 등급 정보를 JavaScript로 가져올 수 있나요?
   예: var memberGrade = '{$member.grade}';

2. 등급별 할인율을 상품 목록/상세 페이지에서 동적으로 표시하려면 
   어떤 변수나 API를 사용하면 될까요?

3. 회원 등급명을 다음과 같이 설정할 수 있나요?
   - '강사' 또는 'instructor'
   - '협회' 또는 'association'
   - 'VIP' 또는 'vip'

답변 부탁드립니다!
```

### 3.2 회원 등급 설정

```
관리자 > 회원 > 회원등급 관리
```

**추천 등급 구조:**

| 등급명 | 할인율 | 비고 |
|--------|--------|------|
| 일반 | 0% | 기본 회원 |
| 강사 | 25% | 공인 강사 |
| 협회 | 30% | 제휴 협회 회원 |
| VIP | 35% | 최우수 고객 |

### 3.3 등급별 가격 표시 코드 예시

메이크샵에서 제공하는 회원 등급 변수가 다음과 같다고 가정:

```html
<!-- 메이크샵 스킨 파일에 추가 -->
<script>
  // 메이크샵 회원 정보를 JavaScript로 전달
  window.MakeshopMemberInfo = {
    isLogin: {$is_login},
    memberId: '{$member_id}',
    memberName: '{$member_name}',
    grade: '{$member_grade}'
  };
</script>

<!-- B2B 시스템 스크립트는 자동으로 위 정보를 읽어옵니다 -->
```

### 3.4 테스트 방법

1. 로그아웃 상태에서 페이지 접속 → 일반 가격만 표시
2. 일반 회원으로 로그인 → 일반 가격
3. 강사 회원으로 로그인 → 25% 할인가 표시
4. 협회 회원으로 로그인 → 30% 할인가 표시

**개발 테스트:**
```javascript
// 브라우저 콘솔에서 실행
DevMode.setTestMember('instructor'); // 강사 회원으로 테스트
location.reload();
```

## 4. 유튜브 영상 연동

### 4.1 유튜브 채널 정보
```
채널: https://www.youtube.com/channel/UCOt_7gyvjqHBw304hU4-FUw
```

### 4.2 상품별 영상 연결 방법

**방법 1: 상품 상세 설명에 추가**

```
관리자 > 상품 관리 > 상품 수정
```

상품 상세 설명 최상단에 다음 형식으로 입력:

```html
<!-- 유튜브 영상 ID: VIDEO_ID_HERE -->
<!-- 타임라인: 0,165,420,720,960 -->
```

**방법 2: 상품 부가정보 활용**

메이크샵 담당자에게 문의:
```
"상품에 유튜브 영상 링크를 연결하기 위해 
부가정보 필드를 추가할 수 있나요?"

필요한 필드:
- youtube_video_id (영상 ID)
- youtube_chapters (타임라인: 0,165,420,720,960)
```

### 4.3 유튜브 API 활용

상세 페이지 (`03-product-detail.html`)에서 이미 구현된 기능:

```javascript
// YouTube IFrame API 자동 로드
// 타임라인 버튼 클릭 시 해당 시간으로 이동
// 재료 목록의 타임라인 배지 클릭 시 영상 이동
```

### 4.4 영상-상품 연결 작업 프로세스

1. **영상 목록 정리**
   - 유튜브 채널의 모든 영상 목록 작성
   - 각 영상의 ID 기록 (예: `dQw4w9WgXcQ`)

2. **주요 장면 타임라인 작성**
   ```
   예시: 압화 입문 가이드
   00:00 - 인트로 및 재료 소개
   02:45 - 꽃 선택과 전처리
   07:00 - 프레싱 툴 사용법
   12:00 - 건조 시간과 보관
   16:00 - 액자 구도 잡기
   ```

3. **상품 페이지에 적용**
   - 각 상품 편집
   - 유튜브 영상 ID 입력
   - 타임라인 정보 입력

## 7. 자주 묻는 질문 (FAQ)

### Q1. 메이크샵에서 JavaScript가 작동하지 않아요.

**A:** 메이크샵 관리자 > 디자인 > 고급설정에서 "외부 스크립트 허용" 옵션을 확인하세요.

### Q2. 회원 등급 정보를 가져올 수 없어요.

**A:** 메이크샵 버전마다 변수명이 다를 수 있습니다. 고객센터에 문의하여 정확한 변수명을 확인하세요.

### Q3. 유튜브 영상이 재생되지 않아요.

**A:** YouTube IFrame API가 로드되었는지 확인하세요. 브라우저 콘솔에서 `YT` 객체가 존재하는지 확인할 수 있습니다.

### Q4. 모바일에서 레이아웃이 깨져요.

**A:** `01-brand-system.css`가 제대로 로드되었는지 확인하세요. 미디어 쿼리가 포함되어 있습니다.

### Q5. B2B 회원 전용 UI가 보이지 않아요.

**A:** `04-b2b-member-system.js`가 로드되었고, 회원 정보가 올바르게 전달되는지 확인하세요. 개발자 도구 콘솔에서 `window.Presco21`이 정의되어 있는지 확인할 수 있습니다.
