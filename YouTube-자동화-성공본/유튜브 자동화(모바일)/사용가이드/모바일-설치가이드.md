# 📱 모바일 유튜브 자동화 설치 가이드

## 🎯 개요
모바일 버전에서는 **최신 유튜브 영상 3개**를 자동으로 표시합니다.
데스크톱과 동일한 Google Apps Script를 사용하므로 추가 설정 없이 바로 사용 가능합니다.

---

## 📋 필요한 파일
```
✅ 모바일-유튜브자동화.html
✅ 모바일-유튜브자동화.css  
✅ 모바일-유튜브자동화.js
✅ youtube-proxy.gs (데스크톱과 동일)
```

---

## 🚀 설치 방법

### 1단계: 메이크샵 관리자 접속
1. 메이크샵 관리자 페이지 로그인
2. **디자인 관리 > 모바일 스킨 편집** 메뉴로 이동

### 2단계: HTML 코드 붙여넣기
1. **메인 페이지(index.html)** 편집 화면 열기
2. `모바일-유튜브자동화.html` 파일 내용 **전체 복사**
3. 기존 HTML 코드를 **완전히 교체** 후 저장

### 3단계: CSS 코드 추가
1. **스타일시트(CSS)** 편집 화면 열기
2. `모바일-유튜브자동화.css` 파일 내용 복사
3. 기존 CSS **하단에 추가** 후 저장

### 4단계: JavaScript 코드 추가
1. **JavaScript** 편집 화면 열기
2. `모바일-유튜브자동화.js` 파일 내용 복사
3. 기존 JS **하단에 추가** 후 저장

### 5단계: Google Script URL 확인
```javascript
// 모바일-유튜브자동화.js 파일에서 이 부분을 확인하세요
var GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
```
- 데스크톱과 **동일한 URL**을 사용합니다
- 이미 데스크톱에서 설정했다면 그대로 사용 가능

## ✨ 주요 특징

### 📱 모바일 최적화
- **1열 레이아웃**: 모바일 화면에 최적화된 세로 배치
- **3개 영상**: 빠른 로딩을 위해 3개만 표시
- **터치 친화적**: 모바일 터치에 최적화된 UI
- **반응형 디자인**: 모든 모바일 화면 크기 지원

### 🎬 자동화 기능
- ✅ 최신 영상 3개 자동 로드
- ✅ 새 영상 업로드 시 자동 갱신
- ✅ 로딩 애니메이션 표시
- ✅ 에러 처리 자동화

## 🔧 문제 해결

### 영상이 표시되지 않는 경우

#### 1. Google Script URL 확인
```javascript
// 올바른 형식인지 확인
var GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/.../exec';
```

#### 2. 브라우저 콘솔 확인
- 모바일 브라우저에서 개발자 도구 열기
- 콘솔에서 에러 메시지 확인
- `[YouTube Mobile]` 로그 메시지 확인

#### 3. 네트워크 연결 확인
- 모바일 데이터 또는 WiFi 연결 상태 확인
- Google Apps Script 접근 가능 여부 확인

### 레이아웃이 깨지는 경우

#### 1. CSS 충돌 확인
```css
/* 기존 CSS와 충돌 시, 우선순위 높이기 */
.mobile-youtube-grid {
    display: flex !important;
    flex-direction: column !important;
}
```

#### 2. jQuery 로드 확인
- 메이크샵 기본 jQuery가 로드되는지 확인
- 보통 자동으로 로드되지만, 안 되면 추가 필요

## 📊 데스크톱 vs 모바일 비교

| 항목 | 데스크톱 | 모바일 |
|------|---------|--------|
| 영상 개수 | 4개 | 3개 |
| 레이아웃 | 2x2 그리드 | 1열 세로 배치 |
| 영상 크기 | 중간 | 전체 너비 |
| 로딩 속도 | 보통 | 빠름 |
