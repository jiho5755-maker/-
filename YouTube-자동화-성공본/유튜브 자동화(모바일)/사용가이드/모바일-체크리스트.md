# ✅ 모바일 유튜브 자동화 체크리스트

## 📱 설치 전 확인사항

- [ ] 데스크톱 버전이 정상 작동 중인가요?
- [ ] Google Apps Script URL을 알고 있나요?
- [ ] 메이크샵 관리자 접근 권한이 있나요?
- [ ] 모바일 스킨 편집 권한이 있나요?

---

## 🚀 설치 단계별 체크리스트

### 1️⃣ HTML 설치
- [ ] 메이크샵 관리자 > 디자인 관리 > 모바일 스킨 편집 접속
- [ ] 메인 페이지(index.html) 열기
- [ ] `모바일-유튜브자동화.html` 전체 복사
- [ ] 기존 HTML 코드 완전히 교체
- [ ] 저장 버튼 클릭
- [ ] 저장 완료 메시지 확인

### 2️⃣ CSS 설치
- [ ] 스타일시트(CSS) 편집 화면 열기
- [ ] `모바일-유튜브자동화.css` 복사
- [ ] 기존 CSS **하단에 추가** (교체 아님!)
- [ ] 저장 버튼 클릭
- [ ] 저장 완료 메시지 확인

### 3️⃣ JavaScript 설치
- [ ] JavaScript 편집 화면 열기
- [ ] `모바일-유튜브자동화.js` 복사
- [ ] 기존 JS **하단에 추가** (교체 아님!)
- [ ] Google Script URL 확인
  ```javascript
  var GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/..../exec';
  ```
- [ ] 저장 버튼 클릭
- [ ] 저장 완료 메시지 확인

## ⚠️ 문제 발생 시 체크리스트

### 영상이 표시되지 않는 경우
- [ ] Google Script URL이 올바른가?
- [ ] 데스크톱 버전은 작동하는가?
- [ ] 모바일 인터넷 연결이 정상인가?
- [ ] 브라우저 콘솔에 에러가 있는가?
- [ ] JavaScript 파일이 제대로 저장되었는가?

### 레이아웃이 깨진 경우
- [ ] CSS 파일이 제대로 저장되었는가?
- [ ] 기존 CSS와 충돌하지 않는가?
- [ ] 브라우저 캐시를 지웠는가?
- [ ] 페이지를 새로고침 했는가?

### 로딩이 느린 경우
- [ ] 영상 개수가 3개로 설정되어 있는가?
- [ ] 모바일 네트워크 속도가 느린가?
- [ ] Google Apps Script가 정상 작동하는가?
