# 📱 메이크샵 모바일 유튜브 자동화

## 🎯 프로젝트 소개

메이크샵 자사몰 **모바일 버전**에 유튜브 채널의 최신 영상 3개를 자동으로 표시하는 시스템입니다.

### ✨ 주요 특징
- 🎬 **완전 자동화**: 유튜브에 새 영상 업로드 시 자동 갱신
- 📱 **모바일 최적화**: 1열 레이아웃으로 모바일에 최적화
- ⚡ **빠른 로딩**: 3개 영상만 표시하여 빠른 로딩
- 🎨 **깔끔한 디자인**: 현대적이고 세련된 UI
- 🔧 **쉬운 설치**: 복사 & 붙여넣기로 간단 설치

### 2️⃣ 설치

모바일 브라우저로 사이트 접속 → 최신 영상 3개 확인 ✅

## ⚙️ 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox 레이아웃, 애니메이션
- **JavaScript (ES5)**: 메이크샵 호환성

### Backend (Proxy)
- **Google Apps Script**: YouTube Data API 프록시
- **YouTube Data API v3**: 영상 데이터 수집

### Libraries
- **jQuery**: DOM 조작, AJAX
- **Swiper**: 배너 슬라이드

## 📊 성능 최적화

### 현재 설정
- ✅ **영상 개수**: 3개 (최적)
- ✅ **Lazy Loading**: iframe 지연 로딩

- 🎯 이미지 최적화: WebP 포맷 사용
- 🎯 정기 점검: 월 1회

## 🐛 문제 해결

### 자주 묻는 질문

#### Q1. 영상이 표시되지 않아요
```javascript
// 1. Google Script URL 확인
var GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/.../exec';

// 2. 데스크톱 버전이 작동하는지 확인
// 3. 브라우저 콘솔에서 에러 확인
```

#### Q2. 레이아웃이 깨져요
```css
/* CSS 우선순위 높이기 */
.mobile-youtube-grid {
    display: flex !important;
    flex-direction: column !important;
}
```

#### Q3. 로딩이 너무 느려요
```javascript
// 영상 개수 줄이기
var MAX_RESULTS = 2;  // 3개 → 2개로 변경
```
