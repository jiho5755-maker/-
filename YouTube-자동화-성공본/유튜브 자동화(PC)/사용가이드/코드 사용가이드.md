## 🔧 문제 해결

### 문제 1: 영상이 안 보여요

**증상:** "최신 영상을 불러오는 중입니다" 메시지만 표시

**해결:**
1. F12 > Console 확인
2. Google Apps Script 배포 상태 확인
   - https://script.google.com
   - 프로젝트 열기
   - 배포 > 배포 관리
   - 상태 확인
3. 배포 URL이 올바른지 확인
4. 10분 대기 후 재시도

---

### 문제 2: 오래된 영상이 보여요

**원인:** 브라우저 캐시

**해결:**
1. Ctrl+F5 (강력 새로고침)
2. 브라우저 캐시 완전 삭제
3. 시크릿 모드로 테스트

---

### 문제 3: Google Script 오류

**증상:** Console에 500 에러

**해결:**
1. Google Apps Script에서 로그 확인
   - 실행 > 실행 로그 보기
2. 채널 ID 확인
3. RSS 피드 접근 가능 여부 확인
4. 스크립트 재배포
