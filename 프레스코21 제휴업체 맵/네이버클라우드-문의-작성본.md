## 📸 스크린샷 준비 가이드

### 1. 콘솔 오류 스크린샷
- F12 누르기
- "콘솔" 탭 클릭
- 전체 화면 캡처 (에러 메시지가 모두 보이도록)

### 2. Network 탭 스크린샷
- F12 누르기
- "네트워크" 탭 클릭
- maps.js 파일이 보이도록 캡처

### 3. 웹 페이지 에러 화면
- 브라우저 전체 화면 캡처
- 에러 메시지 박스가 보이도록

### 4. 네이버 클라우드 플랫폼 콘솔 설정 화면 (선택사항)
- Maps → Application → Pressco21-Maps 페이지
- Web Service URL 섹션이 보이도록 캡처
