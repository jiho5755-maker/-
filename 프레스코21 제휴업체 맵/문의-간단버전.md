## 📸 스크린샷 찍는 방법

### 1. 콘솔 오류 스크린샷
1. 브라우저에서 F12 누르기
2. "콘솔" 탭 클릭
3. 스크린샷 찍기 (에러 메시지가 모두 보이도록)

### 2. Network 탭 스크린샷  
1. F12 누르기
2. "네트워크" 탭 클릭
3. maps.js 파일이 보이도록 스크린샷 찍기

### 3. 웹 페이지 에러 화면
1. 브라우저 전체 화면 보기
2. 에러 메시지 박스가 보이도록 스크린샷 찍기
