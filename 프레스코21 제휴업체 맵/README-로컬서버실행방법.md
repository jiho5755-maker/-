# 로컬 서버 실행 방법

## 🚀 빠른 실행 (가장 쉬움)

### 방법 1: 서버실행.command 파일 사용
1. `서버실행.command` 파일 더블클릭
2. 브라우저가 자동으로 열립니다

### 방법 2: 터미널에서 직접 실행

1. **터미널 열기** (Mac: Spotlight에서 "터미널" 검색)

2. **다음 명령어 실행:**
```bash
cd "/Users/jangjiho/Desktop/커서AI/프레스코21 제휴업체 맵"
python3 -m http.server 8080
```

3. **브라우저에서 다음 주소 접속:**
```
http://localhost:8080/제휴업체-지도-통합본.html
```

---

## ⚠️ 중요 사항

### 올바른 URL 형식:
✅ `http://localhost:8080/제휴업체-지도-통합본.html`
✅ `http://127.0.0.1:8080/제휴업체-지도-통합본.html`

❌ `localhost` (포트 번호 없음 - 404 에러)
❌ `file:///Users/...` (로컬 파일 - 네이버 지도 API 작동 안 함)

---

## 🔍 문제 해결

### 문제: 404 에러 (File not found)

**원인:** URL이 잘못되었거나 서버가 다른 폴더에서 실행됨

**해결:**
1. 터미널에서 올바른 폴더로 이동:
   ```bash
   cd "/Users/jangjiho/Desktop/커서AI/프레스코21 제휴업체 맵"
   ```

2. 파일이 있는지 확인:
   ```bash
   ls -la 제휴업체-지도-통합본.html
   ```

3. 서버 재시작:
   ```bash
   python3 -m http.server 8080
   ```

4. 브라우저 주소창에 정확한 URL 입력:
   ```
   http://localhost:8080/제휴업체-지도-통합본.html
   ```

### 문제: 포트 8080이 이미 사용 중

**해결:** 다른 포트 사용
```bash
python3 -m http.server 8000
```
그 다음 `http://localhost:8000/제휴업체-지도-통합본.html` 접속

### 문제: 서버를 종료하는 방법

터미널에서 **Ctrl + C** 누르기

- [ ] 터미널에서 올바른 폴더에 있음
- [ ] `python3 -m http.server 8080` 명령어 실행

- [ ] 파일명이 정확함: `제휴업체-지도-통합본.html`
- [ ] 네이버 지도가 정상적으로 표시됨
