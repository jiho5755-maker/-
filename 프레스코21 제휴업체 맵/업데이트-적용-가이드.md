# 🎉 제휴업체 지도 업데이트 완료!

## ✅ 추가된 기능 목록

### 1️⃣ **검색 기능** ✅
- 상단에 검색바 추가
- 업체명 실시간 검색
- 검색 결과 URL 저장 (공유 가능)

### 2️⃣ **📍 근처 버튼** ✅
- 현재 위치 기반 정렬
- 거리순 정렬
- 업체 목록에 거리 표시

### 3️⃣ **⭐ 찜 기능** ✅
- 찜한 업체만 보기
- 로컬 저장소에 저장 (브라우저 닫아도 유지)
- 찜 목록 필터링

### 4️⃣ **링크 드롭다운** ✅
- 여러 링크가 있으면 드롭다운으로 표시
- 인스타그램, 페이스북, 블로그, 유튜브 자동 인식
- 링크가 1개면 바로 버튼으로 표시

### 5️⃣ **🚗 길찾기 버튼** ✅
- InfoWindow에 길찾기 버튼 추가
- 네이버 지도 / 카카오맵 선택 가능
- 클릭 시 선택 팝업 표시

### 6️⃣ **📄 상세보기 모달** ✅
- 업체 상세 정보를 모달로 표시
- 큰 이미지 표시
- 전화하기, 길찾기, 찜하기, 공유 버튼
- 카테고리, 협회 정보 포함

### 7️⃣ **🗺️ 지도 스타일 전환** ✅
- 일반 지도 ↔ 위성 지도 전환
- 우측 상단 버튼으로 전환

### 8️⃣ **URL 상태 저장** ✅
- 필터, 검색어를 URL에 저장
- 링크 공유 시 같은 상태로 열림
- 공유 버튼으로 링크 복사

### 9️⃣ **모바일 최적화** ✅
- 검색바 반응형 디자인
- 버튼 크기 모바일 최적화
- 모달 모바일 레이아웃

## 🎮 사용 방법

### 1. 검색
- 상단 검색바에 업체명 입력
- 실시간으로 필터링

### 2. 근처 업체 찾기
1. `📍 근처` 버튼 클릭
2. 위치 권한 허용
3. 거리순으로 자동 정렬
4. 업체 목록에 거리 표시

### 3. 찜하기
1. 마커 클릭 → 상세보기 클릭
2. `⭐ 찜하기` 버튼 클릭
3. `⭐ 찜` 버튼으로 찜한 업체만 보기

### 4. 길찾기
1. 마커 클릭 → `🚗 길찾기` 버튼
2. 네이버 지도 / 카카오맵 선택
3. 앱으로 자동 이동

### 5. 링크 보기
- 링크 1개: 바로 버튼
- 링크 여러 개: `🔗 링크 보기 (N)` 드롭다운

### 6. 상세보기
1. InfoWindow에서 `📄 상세보기` 클릭
2. 큰 이미지, 전체 정보 확인
3. 전화하기, 길찾기, 찜하기, 공유 가능

### 7. 공유하기
1. 상세보기 → `🔗 공유` 버튼
2. 링크가 클립보드에 복사됨
3. 친구에게 전송

### 8. 지도 스타일
- 우측 상단 `🗺️` 버튼 클릭
- 일반 지도 ↔ 위성 지도 전환

## 🐛 문제 발생 시

### 검색이 안 돼요
- 페이지 새로고침 (`Ctrl/Cmd + R`)
- 캐시 초기화 (`Ctrl/Cmd + Shift + R`)

### 근처 버튼이 안 돼요
- 위치 권한 허용 확인
- HTTPS 환경 확인 (GitHub Pages는 자동 HTTPS)

### 링크가 안 보여요
- 구글 시트에서 링크 형식 확인
- 구분자: `,` 또는 `|` 또는 줄바꿈

### 모달이 안 열려요
- 브라우저 콘솔 에러 확인 (`F12`)
- JavaScript 활성화 확인
