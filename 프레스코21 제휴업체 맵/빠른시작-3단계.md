## 🎮 새 기능 사용법 (빠른 튜토리얼)

### 🔍 검색하기
```
1. 상단 검색바 클릭
2. 업체명 입력 (예: "도자기")
3. 실시간 필터링 확인
```

### 📍 근처 업체 찾기
```
1. "📍 근처" 버튼 클릭
2. 위치 권한 허용
3. 거리순 정렬 확인
4. 업체 목록에서 거리 확인
```

### ⭐ 찜하기
```
1. 마커 클릭
2. "📄 상세보기" 클릭
3. "⭐ 찜하기" 클릭
4. 상단 "⭐ 찜" 버튼으로 찜 목록만 보기
```

### 🚗 길찾기
```
1. 마커 클릭
2. "🚗 길찾기" 클릭
3. 네이버 지도 / 카카오맵 선택
4. 앱으로 자동 이동
```

### 🔗 공유하기
```
1. 마커 클릭 → 상세보기
2. "🔗 공유" 클릭
3. 링크가 자동으로 복사됨
4. 친구에게 전송
```

## ⚠️ 문제 해결 (1분 진단)

### 문제: 지도가 안 보여요
```
✅ 네이버 클라우드 플랫폼 콘솔 확인
✅ URL 등록 확인: https://jiho5755-maker.github.io
✅ ncpKeyId 값 확인
```

### 문제: 마커가 안 보여요
```
✅ Google Apps Script 배포 확인
✅ doGet() 함수 실행 확인
✅ JSON API URL 확인
✅ 구글 시트에서 '승인' 상태 확인
```

### 문제: 업데이트가 반영 안 돼요
```
✅ GitHub Actions 완료 확인
✅ 캐시 초기화 (Ctrl/Cmd + Shift + R)
✅ 시크릿 모드에서 확인
```

### 문제: 근처 버튼이 안 돼요
```
✅ HTTPS 환경 확인 (http는 위치 API 불가)
✅ 브라우저 위치 권한 확인
✅ 모바일 위치 서비스 활성화
```
