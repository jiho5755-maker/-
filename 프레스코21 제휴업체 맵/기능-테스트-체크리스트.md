## 기능 테스트

### 🔍 검색 기능
- [ ] 검색바에 업체명 입력 시 필터링
- [ ] 검색 결과가 즉시 반영
- [ ] 검색어가 URL에 저장

### 📍 근처 버튼
- [ ] 버튼 클릭 시 위치 권한 요청
- [ ] 위치 허용 시 거리순 정렬
- [ ] 업체 목록에 거리 표시 (예: "1.2km")
- [ ] 버튼 활성화 상태 표시

### ⭐ 찜 기능
- [ ] 상세보기에서 찜하기 버튼 동작
- [ ] 찜 목록이 로컬 저장소에 저장
- [ ] 찜 버튼 클릭 시 찜한 업체만 필터링
- [ ] 페이지 새로고침해도 찜 목록 유지

### 🔗 링크 드롭다운
- [ ] 링크 1개일 때 바로 버튼 표시
- [ ] 링크 여러 개일 때 드롭다운 표시
- [ ] 인스타그램 아이콘 자동 표시
- [ ] 블로그 아이콘 자동 표시
- [ ] 유튜브 아이콘 자동 표시
- [ ] 드롭다운 클릭 시 메뉴 열림

### 🚗 길찾기
- [ ] InfoWindow에 길찾기 버튼 표시
- [ ] 클릭 시 네이버/카카오 선택 팝업
- [ ] 네이버 지도 링크 정상 작동
- [ ] 카카오맵 링크 정상 작동

### 📄 상세보기 모달
- [ ] InfoWindow에서 상세보기 버튼 클릭
- [ ] 모달이 부드럽게 나타남
- [ ] 이미지가 크게 표시
- [ ] 전화하기 버튼 동작
- [ ] 길찾기 버튼 동작
- [ ] 찜하기 버튼 동작
- [ ] 공유 버튼 동작 (링크 복사)
- [ ] X 버튼으로 닫기
- [ ] 배경 클릭으로 닫기

### 🗺️ 지도 스타일
- [ ] 우측 상단 버튼 표시
- [ ] 클릭 시 위성 지도로 전환
- [ ] 다시 클릭 시 일반 지도로 전환
- [ ] 버튼 아이콘 변경 (🗺️ ↔ 🛰️)

### 🔗 URL 상태 저장
- [ ] 필터 변경 시 URL 업데이트
- [ ] 검색 시 URL 업데이트
- [ ] URL 복사 후 새 탭에서 열기
- [ ] 같은 상태로 페이지 열림
