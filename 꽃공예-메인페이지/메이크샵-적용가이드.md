# 🌸 메이크샵 전용 - 꽃 공예 브랜드 CSS 적용 가이드

## ⚡ 빠른 적용 (3분)

### 1단계: CSS 업로드
메이크샵 관리자 → 디자인 관리 → 소스 편집

**파일**: `메이크샵-최적화.css`를 열어서 **전체 복사**

메인 CSS 파일 맨 아래에 **붙여넣기** 후 저장

### 2단계: HTML에 폰트 추가
메인 HTML 파일 `<head>` 부분에 추가:

```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Serif+KR:wght@400;600;700&family=Noto+Sans+KR:wght@400;500;600;700&display=swap" rel="stylesheet">
```

### 3단계: 완료!
**Ctrl + Shift + R** (강력 새로고침)

## 🔧 색상 변경

메이크샵에서 직접 색상을 바꾸려면 CSS 상단의 값을 수정:

```css
:root {
    --color-primary: #7B8E7E;      /* ← 메인 색상 */
}
```

### 다른 색으로 변경 예시

**핑크 톤**
```css
--color-primary: #D4A5A5;
```

**블루 그린**
```css
--color-primary: #6B8E8E;
```

**라벤더**
```css
--color-primary: #A78BB7;
```

## 🐛 문제 해결

### Q. 색상이 안 바뀌어요
**A.** 브라우저 캐시 삭제
- Chrome: `Ctrl + Shift + Delete`
- 또는 `Ctrl + Shift + R` (강력 새로고침)

### Q. 폰트가 안 바뀌어요
**A.** Google Fonts 링크 확인
- `<head>` 태그 안에 있어야 함
- 링크가 정확한지 확인

### Q. 레이아웃이 깨져요
**A.** CSS를 잘못 붙여넣었을 가능성
- 백업한 CSS로 복원
- 다시 천천히 붙여넣기

### Q. 일부만 바뀌어요
**A.** CSS 우선순위 문제
- `!important` 추가:
```css
background-color: #FAF9F6 !important;
```
