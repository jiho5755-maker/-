# 🌸 메이크샵 전용 - 꽃 공예 브랜드 CSS

> 메이크샵에서 바로 사용 가능한 CSS 파일입니다.

## 📦 파일 구성

```
✅ 메이크샵-최적화.css        ← 이 파일만 사용하세요!
📖 메이크샵-적용가이드.md      ← 필독!
🎯 상단메뉴-가이드.md          ← 메뉴 스타일 가이드
```

## 🎨 변경사항

### 색상
- 파란색 → **세이지 그린 (#7B8E7E)**
- 순백색 → **웜 화이트 (#FAF9F6)**

### 디자인
- 모든 모서리 8px 둥글게
- 부드러운 Hover 애니메이션
- 명조체 제목 (Noto Serif KR)

### 상단 메뉴 ⭐ NEW
- **Hover**: 세이지 그린 + 배경 강조
- **선택됨**: 세이지 그린 배경 + 흰색 글자 (명확!)
- **서브메뉴**: 부드러운 그림자

---

## ⚡ 3분 안에 적용하기

### 1. CSS 복사
`메이크샵-최적화.css` 파일 열어서 전체 복사 (Ctrl+A, Ctrl+C)

### 2. 메이크샵에 붙여넣기
```
메이크샵 관리자 
→ 디자인 관리 
→ 소스 편집 
→ CSS 파일 맨 아래에 붙여넣기
```

### 3. 폰트 추가
HTML `<head>` 태그에 추가:
```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Serif+KR:wght@400;600;700&family=Noto+Sans+KR:wght@400;500;600;700&display=swap" rel="stylesheet">
```

### 4. 저장 후 새로고침
**Ctrl + Shift + R**

끝! 🎉

## 💡 색상 바꾸기

CSS 파일 맨 위에서:

```css
:root {
    --color-primary: #7B8E7E;    /* ← 여기를 바꾸면 전체 변경 */
}
```

### 다른 색 예시
- 핑크: `#D4A5A5`
- 블루그린: `#6B8E8E`
- 라벤더: `#A78BB7`
