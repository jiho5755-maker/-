# 🌸 메이크샵 전용 - 상단 메뉴 스타일 가이드

## 📋 상단 네비게이션 개선사항

### ✅ 적용된 효과

#### 1️⃣ 기본 메뉴
- 색상: 진한 녹색 (#2F3E32)
- 폰트: 중간 굵기 (500)

#### 2️⃣ Hover (마우스 올렸을 때)
```
✓ 색상: 세이지 그린 (#7B8E7E)
✓ 글자: 굵게 (700)
✓ 배경: 연한 세이지 그린 (10% 투명도)
```

#### 3️⃣ Active/선택됨 (현재 페이지)
```
✓ 배경: 세이지 그린 (#7B8E7E)
✓ 글자: 흰색 (명확하게 보임!)
✓ 굵기: 굵게 (700)
✓ 모서리: 살짝 둥글게 (4px)
```

#### 4️⃣ 서브메뉴 (드롭다운)
```
✓ 배경: 웜 화이트 (#FAF9F6)
✓ 테두리: 연한 회색
✓ 그림자: 부드러운 세이지 그린 톤
✓ Hover: 아이보리 배경 + 왼쪽으로 살짝 이동
```

## 🎨 시각적 효과

### Before (기존)
```
[홈]  [상품]  [이벤트]  [고객센터]
     ↑ 선택해도 구분 안됨
```

### After (개선)
```
[홈]  [상품]  [이벤트]  [고객센터]
      ━━━━
     배경색 + 흰글자 = 명확!
```

---

## 💡 커스터마이징

### 메뉴 색상 바꾸기

CSS 파일에서:

```css
/* Hover 색상 변경 */
#header nav ul li:hover > a {
    color: #7B8E7E;  /* ← 원하는 색으로 */
}

/* 선택된 메뉴 배경색 변경 */
#header nav ul li.active > a {
    background-color: #7B8E7E;  /* ← 원하는 색으로 */
}
```

### 다른 색 예시

**핑크 톤**
```css
color: #D4A5A5;
background-color: #D4A5A5;
```

**블루 그린**
```css
color: #6B8E8E;
background-color: #6B8E8E;
```

## 📱 반응형

### PC (768px 이상)
- 가로 배치 메뉴
- Hover 시 서브메뉴 드롭다운

### 모바일 (767px 이하)
- 세로 배치 메뉴
- 각 메뉴 클릭 시 전체 배경 변경
- 명확한 구분선

## ⚠️ 주의사항

1. **메뉴 클래스명 확인**
   - 메이크샵마다 클래스명이 다를 수 있음
   - `.gnb`, `.main-menu`, `nav.navigation` 등

2. **!important 사용**
   - 기존 스타일과 충돌 방지
   - 필요한 곳에만 사용

3. **모바일 테스트 필수**
   - PC와 모바일 모두 확인
